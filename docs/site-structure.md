**Публичная часть**

**Главная страница**

Содержит слайдер с примерами выполненных работ, общую информацию о компании, навигацию по сайту и призыв к действию. Все элементы объединены в одной странице без выделения дополнительных разделов.

**Страница каталога услуг**

Отображает список всех ремонтных услуг с возможностью поиска и фильтрации. Детальная информация об отдельной услуге открывается внутри этой же страницы без перехода на отдельный URL.

Страница детальной информации об услуге

Разворачивается внутри каталога и показывает название услуги, цену, фото, время выполнения работы и дополнительные условия.

**Страница входа и регистрации**

Единая страница с переключением между формой авторизации и формой создания нового аккаунта.
Личный кабинет (доступен после авторизации)

**Личный кабинет пользователя**

Единая страница, на которой пользователь может создать новую заявку на ремонт, просмотреть список своих текущих и завершённых заявок, а также выйти из системы. Дополнительных страниц настроек профиля не предусмотрено.

**Административная часть**

**Админ-панель**
Единая страница для управления сайтом. Администратор может создавать новые услуги, просматривать список существующих услуг, удалять услуги и просматривать заявки, оставленные пользователями. Все функции управления реализованы в рамках одного интерфейса без разделения на подстраницы.

Итого: 6 страниц
