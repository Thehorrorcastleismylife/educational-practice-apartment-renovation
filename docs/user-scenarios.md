
# Пользователь:
Главная → Каталог → Детальная страница → Форма записи → Личный кабинет

# Гость:
Главная → Каталог → Детальная страница → Страница регистрации →  Форма записи → Личный кабинет

# Администратор:
Админ-панель → создание услуги → удаление услуги
Админ-панель → просмотр заявок пользователя → отправить данные для связи мастеру по ремонту и пользователю

# Огранчения
## Отсутствует онлайн-оплата.
Пользователь не может оплатить услугу через сайт — оплата происходит вне системы, напрямую с мастером или оператором. 

## Отсутствует система рейтингов и отзывов. 
Пользователи не могут оставлять оценки мастерам или публиковать отзывы о выполненных работах.

## Отсутствует личный профиль мастера.
нет карточек конкретных мастеров с информацией об их специализации, опыте и выполненных работах.
